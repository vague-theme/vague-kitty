## Usage

1. Clone the repository. For example into `~/.config/kitty`:
   ```sh
   git clone https://github.com/vague-theme/vague-kitty ~/.config/kitty/vague
   ```

2. Add an include directive to your `kitty.conf`:
   ```conf
   include vague/vague.conf
   ```
